# Real-Estate-Price-Prediction
Real Estate Price Prediction is a predictive analytics project that estimates property prices using machine learning algorithms. It includes data preprocessing, exploratory analysis, feature engineering, model training, and evaluation to build a data-driven price prediction system using Python and machine learning libraries.

# Technologies Used
. Python
. Pandas
. NumPy
. Matplotlib

# Workflow
Data Cleaning
Exploratory Data Analysis
Feature Engineering
Model Training
Model Evaluation

# Machine Learning Models
Linear Regression
Decision Tree
Random Forest

# Results
Model performance evaluated using accuracy metrics.
